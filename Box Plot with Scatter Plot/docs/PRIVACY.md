# Privacy Statement – Box Plot with Jittered Points (Spotfire Mod)

This Spotfire visualization mod does not collect, transmit, or store any user, usage, or data content outside the Spotfire client environment.

## Data handling
- No outbound network requests are performed beyond those initiated by Spotfire itself.
- No analytics, telemetry, tracking pixels, beacons, or cookies are used by the mod.
- No localStorage/sessionStorage persistence is used for data; the mod uses Spotfire ModProperties to persist visualization configuration within Spotfire.
- The mod operates entirely within the Spotfire sandbox and uses only publicly documented Spotfire Mods APIs.

## Third-party code
- The runtime bundle is self-contained and includes no third-party libraries that collect or transmit data.
- Any future third-party components will be listed in THIRD_PARTY_NOTICES.txt and declared in the submission form.

## Scope of data usage
- The mod reads data solely from the Spotfire data view assigned via visualization axes (X, Y, Details, Trellis) and renders visual elements locally in the client.
- The mod does not share, export, or upload any data to external services.
